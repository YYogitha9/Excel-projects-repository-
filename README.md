
# 📊 Excel Data Analysis Project – Bike Sales Dataset

## 📘 Overview
This project was completed as part of a Data Technician Bootcamp, focusing on core Excel skills for data analysis. Using retail and bike sales datasets, the project demonstrates how to clean, transform, analyse, and visualise data to extract meaningful insights.

---

## 🧰 Skills & Tools Used

### 📌 Excel Functions
- `SUM` and `SUMIF` – Calculated total sales and conditional totals
- `AVERAGE` and `AVERAGEIF` – Derived average metrics across datasets
- `DATE`, `MONTH`, `YEAR` – Extracted and analysed time-based data
- `UNIQUE` – Identified distinct values within datasets
- `VLOOKUP` – Retrieved related data across tables
- `SWITCH` – Categorised sales volumes into meaningful groups (High, Medium, Low)

- Product demand varied significantly by region, highlighting opportunities for targeted marketing strategies.
